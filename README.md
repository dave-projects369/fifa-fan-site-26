# WorldCupGolazo ⚽🏆

The ultimate fan destination for everything FIFA World Cup.

## Pages
- `index.html` — Homepage
- `pages/simulator.html` — Bracket Simulator (all 48 teams, 2026 format)
- `pages/arena.html` — World Cup Arena (anthems + stories)
- `pages/history.html` — History Vault (coming soon)
- `pages/travel.html` — Travel Guide (coming soon)
- `pages/quiz.html` — WC Quiz (coming soon)

## Deploy on Vercel (free)

1. Push this folder to your GitHub repo (dave-projects369)
2. Go to vercel.com → New Project → Import from GitHub
3. Select this repo → Deploy
4. Your site is live instantly at yourname.vercel.app
5. Optional: connect a custom domain like worldcupgolazo.com

## Tech Stack
